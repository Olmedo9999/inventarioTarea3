Sistema web desarrollado para una tienda tecnológica con el objetivo de mejorar el control de productos, proveedores, ventas e inventario mediante una plataforma centralizada.


## Tecnologías Utilizadas
- HTML5
- CSS3
- PHP
- MySQL
- XAMPP


## Funcionalidades Principales
- Inicio de sesión con roles.
- Gestión de usuarios.
- Gestión de productos.
- Gestión de proveedores.
- Registro de ingresos.
- Registro de ventas.
- Historial de movimientos.
- Alertas de stock.


El sistema permite optimizar la administración del inventario, mejorar el control de existencias y facilitar la gestión operativa de la tienda.
